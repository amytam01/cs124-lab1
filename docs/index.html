Marking tasks as completed:

By clicking on either the checkbox on the left or the text itself, the user can mark a task as complete. Here, the user marks the task "Call Mom" completed.

Editing and deleting tasks:

The large "Delete All Complete Tasks" button does exactly what it says. Here, the user clicks it to clear three tasks from the list.

Deleting all tasks:

By clicking on the pencil icon, the user can edit the name of a task directly. Similarly, the user can click on the trash can to delete the respective task. Here, the user renames the task "Text John" to "Text John about bank statements" and deleted the task "Eat Lunch."

Adding new tasks to empty/non-empty lists:

The form at the bottom of the screen allows the user to add tasks to the to-do list. Here, the user adds a task to an empty list, then adds another task.
